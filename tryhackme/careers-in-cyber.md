**Plataforma:** TryHackMe

## O que é essa sala

Visão geral das diferentes carreiras em cibersegurança, explicando os papéis existentes, o que cada profissional faz no dia a dia e qual é o caminho de entrada para cada área.

---

## Carreiras apresentadas

| Cargo | O que faz |
|-------|-----------|
| Penetration Tester | Testa sistemas buscando vulnerabilidades de forma autorizada |
| Red Teamer | Simula ataques reais para testar a defesa da empresa |
| Security Analyst (Blue Team) | Monitora e defende sistemas contra ataques |
| Security Engineer | Projeta e mantém sistemas de segurança |
| Malware Analyst | Analisa vírus e softwares maliciosos |
| Digital Forensics | Investiga incidentes e coleta evidências digitais |
| Incident Responder | Responde a ataques em tempo real |

---

## O que aprendi

- Existe muito mais do que apenas "hacker" dentro da cibersegurança
- Blue Team (defensivo) tem mais vagas júnior do que Red Team (ofensivo)
- O caminho mais acessível para iniciantes é começar como Security Analyst / SOC
- Todas as carreiras exigem base sólida em redes, Linux e sistemas operacionais

---

## Decisão após essa sala

Confirmei que quero seguir o caminho de segurança ofensiva (Penetration Tester) a longo prazo, mas vou começar pelo SOC / Blue Team para ganhar experiência prática primeiro — que é onde estão as vagas júnior.
